## ターミナル出力用 絵文字一覧表

| カテゴリ | 絵文字 | 説明 | 用途の例 |
| :--- | :--- | :--- | :--- |
| ✅ 成功/完了 | ✅ | 処理が成功し、タスクが完了したことを示します。 | `✅ 接続成功`、`✅ 処理完了` |
| | ✔️ | シンプルなチェックマーク。ログの項目単位での完了。 | `✔️ 設定確認済み` |
| | 🎉 | 処理の終端や、大きなマイルストーン達成を祝う。 | `🎉 全てのテストに合格` |
| ❌ 失敗/警告 | ❌ | 処理が失敗、または操作が拒否されたことを示します。 | `❌ 認証失敗`、`❌ 致命的なエラー` |
| | ⚠️ | 潜在的な問題、警告、非推奨の操作。 | `⚠️ パスワードが期限切れ` |
| | 🚨 | 緊急性、セキュリティアラート、または重大なエラー。 | `🚨 サーバーダウン` |
| ➡️ 処理/方向 | ➡️ | メッセージを送信中、または処理が次に進む方向。 | `➡️ クライアントHello送信` |
| | ⬅️ | データや応答を受信中、または前のステップに戻る方向。 | `⬅️ サーバー応答受信` |
| | 🔄 | 処理の再試行、同期、またはループ操作。 | `🔄 キャッシュを更新中` |
| | ⏳ | 時間がかかる処理、待機が必要な状態。 | `⏳ データベースに接続中` |
| ℹ️ 情報/ヒント | 💡 | 解決策、ヒント、重要な学習ポイント。 | `💡 修正方法の提案` |
| | ℹ️ | 一般的な情報、ログレベルINFOに相当。 | `ℹ️ 設定ファイル読み込み` |
| | ❓ | ユーザーからの入力を待っている、または不明な状態。 | `❓ 続行しますか？` |
| | 💬 | ユーザーとの対話、ログ内のコメント。 | `💬 ユーザー入力を待っています` |
| ⚙️ システム/ネットワーク | ⚙️ | システム設定、構成変更。 | `⚙️ NETCONFサブシステム起動` |
| | 🔌 | 接続の状態、ネットワークケーブル。 | `🔌 リンクアップ` |
| | 🌐 | グローバルなネットワーク、インターネット関連。 | `🌐 外部接続を確認` |
| | 💻 | クライアント側、ローカル環境の操作。 | `💻 スクリプトを実行` |
